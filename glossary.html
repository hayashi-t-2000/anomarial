世界観

SEC-01 / WORLDVIEW
DESCRIPTION

異現実を収容、管理、研究する極秘研究所。マンションの一室から地下の大施設まで大小さまざまな支部が世界各地に配置されている。設立は20年前。前身となる組織はそれぞれの国で抱えていたが、異現実統合管理協定により全て異現実収容研究所へと統合された。

所属スタッフは合計で2000を超える。

本部は南極大陸地下にあるUndertica(アンダルティカ)中央研究所。

権限は以下の通りになる
Lv5・・・議会
Lv4・・・支部長、ICD指揮官
Lv3・・・博士、ICD班長
Lv2・・・助手、研究員、調査班(I班)、回収班(C班)、処理班(D班)
Lv1・・・測定員、事務スタッフ
Lv0・・・被験者

RELATED: 異現実 / 収容プロトコル / 来栖支部
DESCRIPTION

異現実研究所に於ける対話可能な異現実の専門支部。
測定員制度を採用しており、小規模ながら異現実研究における重要な役割を担っている。現所長の■■博士は本部勤務時にAC-■■■との対話を【成功】させ、以降対話可能な異現実の専門家として組織に尽力している。
設立当初、所長を務めていた来栖博士は現在AC-■■■の影響で中央研究所にて収容されている。

RELATED: 測定員 / 異現実データベース
DESCRIPTION

我々が認識する「現実」は単一ではなく、複数の層（レイヤー）として存在するという理論に基づく概念。通常の人間は第一層（一次現実）にのみ属しているが、異現実はこの層を超えた領域から干渉してくるとされている。

現実層の詳細な構造については研究途上であり、現時点では■■■■層の存在が確認されている。

RELATED: 異現実 / 侵食現象
DESCRIPTION

異現実が一次現実に影響を及ぼす現象の総称。軽度の侵食では空間的な違和感・知覚の歪みが生じる程度だが、重度の侵食では物理法則の局所的な崩壊が観測されている。

収容施設内では侵食現象を一定範囲に抑制する設備が稼働しているため、測定員の安全は保障されている。ただし収容区画内では軽度の侵食現象を体感する場合がある。これは正常な範囲内である。

役職・人員

SEC-02 / PERSONNEL
DESCRIPTION

異現実解析に置いて研究所側ではない人材を測定に派遣する制度。研究員に対して敵対視、或いは興味を示さない異現実に対して解析や測定が必要になった際はダークウェブ等に痕跡を残し有志を募る。

軽度の記憶処理を施す必要アリ。

測定員の大まかなタスクは下記の三点。他異現実の性質によっては変動する。
・友好カテゴリの特定
・危険カテゴリの特定
・特別処置

RELATED: キャスト / クリアランスレベル
DESCRIPTION

イベント内で施設スタッフや異現実を演じる運営側のメンバー。施設職員・研究者・警備員など複数の役職が存在する。

イベント中、キャストは担当する役職・キャラクターとして行動する。プロフィールの確認やフレンド申請など、キャストのVRChatアカウントへの個人的なアクセスはイベント中禁止されている。

RELATED: 測定員 / 異現実
DESCRIPTION

施設内における情報・区画へのアクセス権限を示す等級制度。レベル0（一般市民相当）からレベル5（上位機関直属）まで段階が存在する。

測定員はVISITORとして扱われ、一般公開資料へのアクセスのみが許可されている。施設内の機密区画への立ち入りには対応するレベルのIDカードが必要となる。

異現実に関する用語

SEC-03 / ANOMAREAL TERMINOLOGY
DESCRIPTION

本施設が収容・研究の対象とする未知の存在・現象・物体の総称。現行の物理・生物・化学的知見では説明できない性質を持つものとして定義される。

異現実は個体ごとに固有のACナンバー（例：AC-009）が付与され、データベースに登録される。各異現実はカテゴリコードにより脅威度・性質が分類されている。

RELATED: ACナンバー / カテゴリコード / 収容プロトコル
DESCRIPTION

各異現実に付与される固有の識別番号。「AC-」の後に3桁の数字が続く形式（例：AC-009、AC-121）。登録順に番号が割り振られるが、欠番も存在する。

AC-999は施設内に特別に設置された収容区画であり、通常の異現実とは異なる扱いがされている。詳細はクリアランスレベル2以上で閲覧可能。

RELATED: 異現実 / カテゴリコード
DESCRIPTION

異現実の性質・脅威度を示す2文字のコード。アルファベット2文字をスラッシュで区切った形式（例：F/H、N/U）で表記される。

前半は存在様式（F: 固定型 / N: 流動型 / O: 観測依存型）、後半は危険度（H: 高危険 / P: 準危険 / U: 未分類）を示す。ただし分類基準の詳細は機密事項につき公開されていない。

RELATED: 異現実 / ACナンバー / 収容プロトコル
DESCRIPTION

各異現実を安全に維持・管理するために定められた手順書。異現実ごとに個別のプロトコルが存在し、収容条件・接触規則・緊急時対応が記載されている。

測定員が直接プロトコルにアクセスすることはできないが、担当キャストから必要事項の説明を受ける場合がある。プロトコル違反は施設全体の安全を脅かす可能性がある。

RELATED: 異現実 / カテゴリコード / 侵食現象

イベント用語

SEC-04 / EVENT TERMINOLOGY
DESCRIPTION

本イベントの形式。参加者（測定員）がキャストと直接会話・対話を行いながら進行するロールプレイ型の体験型イベント。台本は存在せず、対話の内容によって展開が変化する。

参加者がRP（ロールプレイ）に積極的に参加することで、よりイベントを楽しめる。ただし強制ではなく、測定員として施設を見学するスタンスでも参加可能。

RELATED: 測定員 / キャスト
DESCRIPTION

イベントの世界観・フィクション性を破壊する発言の総称。「これってイベントだよね」「VRChatだから」「中の人は〜」といった発言が該当する。

メタ発言は他の参加者・キャストの体験を損なう行為として禁止されている。世界観に沿った発言・行動を心がけること。

RELATED: 対話型アトラクションRP / 注意事項
DESCRIPTION

VRChatの安全設定機能。シールドレベルを上げると他プレイヤーのアバター・エフェクトが非表示になる。イベント中はシールドレベルを0に設定することで、キャストのアバターやギミックを正しく体験できる。

設定変更はVRChat内のメニュー →「Safety」から行える。イベント開始前に必ず確認すること。
