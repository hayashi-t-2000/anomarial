異現実について

SEC-01 / WORLDVIEW
DESCRIPTION

出自不明の存在。有機物、無機物、生物、或いは空間。様々な形や特徴をそれぞれ有しているがその構造や生態は科学的に解明できないことの方が多い。
人類の文化に準じた姿も確認されているため、何者かの手によって作られているのではないかとも考えられている。

起源は人類の記録媒体誕生の時代に遡る。古代の壁画や出土品から異現実と思われる記録が残されており、当研究所では宇宙人や心霊などのオカルトとされてきたものは全て異現実とカテゴライズし調査する。

特に危険性の高い異現実や人類に有益をもたらす異現実に関しては一国家の占有を阻止するため調査後、各国の収容所へ分配され情報を共有する。
発見順からAC(ｱﾉﾏﾘｱﾙｺｰﾄﾞ)-000(番号)と表記、呼称し友好性、危険性をカテゴライズする。

DESCRIPTION

既に人類に対して適合した異現実の呼称。我々人類では処理や分析ができないので存在すら確かめられていない。

中央の記録を参照：AC-000の記述を発見。詳細は情報操作により隠蔽されている。

異現実収容研究所について

SEC-02 / ANOMAREAL LAB
DESCRIPTION

異現実を収容、管理、研究する極秘研究所。マンションの一室から地下の大施設まで大小さまざまな支部が世界各地に配置されている。設立は20年前。前身となる組織はそれぞれの国で抱えていたが、異現実統合管理協定により全て異現実収容研究所へと統合された。

所属スタッフは合計で2000を超える。

本部は南極大陸地下にあるUndertica(アンダルティカ)中央研究所。

権限は以下の通りになる
Lv5・・・議会
Lv4・・・支部長、ICD指揮官
Lv3・・・博士、ICD班長
Lv2・・・助手、研究員、調査班(I班)、回収班(C班)、処理班(D班)
Lv1・・・測定員、事務スタッフ
Lv0・・・被験者

DESCRIPTION

異現実研究所に於ける対話可能な異現実の専門支部。
測定員制度を採用しており、小規模ながら異現実研究における重要な役割を担っている。現所長のミタライ博士は本部勤務時にAC-■■■との対話を【成功】させ、以降対話可能な異現実の専門家として組織に尽力している。
設立当初、所長を務めていた来栖博士は現在AC-■■■の影響で中央研究所にて収容されている。

DESCRIPTION

異現実研究における核ともいえる場所。世界中様々な研究機関から集められたプロフェッショナルが知恵を出し合って研究を進めている。
5つのエリアで区画分けされており、エリアを統括する博士はLv4の権限を付与される。

中央研究所を起点に各支部へのテレポートが可能。(AC-■■■を参照)

測定員制度について

SEC-03 / MEASUREMENT
DESCRIPTION

異現実解析に置いて研究所側ではない人材を測定に派遣する制度。研究員に対して敵対視、或いは興味を示さない異現実に対して解析や測定が必要になった際はダークウェブ等に痕跡を残し有志を募る。

軽度の記憶処理を施す必要アリ。

測定員の大まかなタスクは下記の三点。他異現実の性質によっては変動する。
・友好カテゴリの特定
・危険カテゴリの特定
・特別処置

DESCRIPTION

適切な収容管理のためにカテゴリを選別する必要がある。友好/危険の頭文字表記で記録する。O/P(敵対かつ危険)は抹消対象とする。

友好カテゴリ
Opponents・・・敵対、攻撃的
Neutral・・・中立
Friendly・・・友好、献身的

危険カテゴリ
Perilous・・・危険かつ制御不能、強大な力
Useful・・・危険かつ制御可能、有用可能
Harmless・・・無害、有用性なし

DESCRIPTION

仮収容している異現実に対してどのような処置を行うか、Lv3以上の判断が必要になる。特別処置は大きく分けると3種類、現状維持を含めると4つの選択肢がある。

〇解放措置・・・異現実に協力者として研究所内の自由を与える
〇収容維持・・・現状の収容方法で問題ない。
〇トラウマ処理・・・記憶改ざんや刺激による精神コントロール
〇処分・・・収容室に設置されている装置による処分
